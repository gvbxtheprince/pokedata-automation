# ⚡ PokéData Automation (Node.js)

Script de automação para extração e processamento de dados competitivos de Pokémon (VGC).

## 🛠️ Tecnologias
- **Node.js** (Runtime)
- **Fetch API** (Consumo de dados)
- **FS Promises** (Persistência de dados)

## 🚀 Como funciona
O script realiza requisições assíncronas em paralelo para a **PokeAPI**, processa o JSON de resposta e formata os dados em um relatório técnico em Markdown. Este é um exemplo prático de um fluxo **ETL (Extract, Transform, Load)**.

## 📦 Como rodar
1. `npm install`
2. `node index.js`
